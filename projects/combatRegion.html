A remastering of the arcade classic Battle Zone with improved graphics

Blow up as many tanks as you want in this early style first-person shooter about maneuvering your tank around a battlefield. Aim for the high score as the difficulty amps up with every kill! To see more about the game, check out the showcase on this page.

This game has all the core mechanics of the original, with one main extra feature. Because the game uses rasterized graphics through WebGL, we can now apply textures and shading to our objects! Pressing the exclamation mark (!) will toggle the new set of graphics.

This was made as the final project for my CSC 561 (Computer Graphics) class at NC State University.

Remarks

  • This was my first time programming a full game in Javascript without a pre-existing engine.
  • All the backend for this game, including storage of model data, shader management, and transform representation is designed and implemented by yours truly.
  • All assets were found online for free or made by hand.
  • The showcase video was made in 3 hours using DaVinci Resolve.
