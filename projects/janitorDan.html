A top-down grid-based puzzle game set in a futuristic lab.

In this game, an unknown observer subjects you to various tests and experiments, trying to examine your puzzle-solving ability. The puzzles are physically based, revolving around using simple movement to manipulate your surroundings and clear a path to the exit.

This was made as a project for my CSC 281 (Game Design) class at NC State University. The whole game was made using the Puzzlescript engine.

Remarks

  • My first browser-based game!
  • Probably some of the puzzle design I've ever done.
  • Though a relatively simple rules-based engine, Puzzlescript can be pushed to some incredible limits, making even animations and easter eggs possible if you're clever enough.
