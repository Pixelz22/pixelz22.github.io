A fully implemented Discord Bot that allows users to play Liar's Dice with each other through an embed-based GUI.

This bot (nicknamed "LoRiggio") allows server members to start, join, play, and customize games of Liar's Dice, a game made popular by the famous scene from Pirates of the Caribbean: Dead Man's Chest. Players can interact with the bot through intuitive buttons and dropdowns, or special commands for more complicated tasks. LoRiggio responds with custom emojis and embeds to present a polished UI for players, using Discord's special ephemeral messages to keep each player's hand private.

Remarks

  • Uses asynchronous command handling.
  • Commands are restricted by server permissions and game state to prevent misuse.
  • Supports 4 different gamemodes and customizable rules.
  • Each text channel can run its own concurrent game.
